Design a complete UI prototype for a modern web application called **StudyFlow**, a study planner website for university students. The design should be clean, minimal, and productivity-focused. Use a modern academic style with soft colors (light background, blue or purple accent colors), rounded cards, clear typography, and intuitive navigation. The design should prioritize usability and clarity.

Create a **desktop web interface** and include the following pages:

1. LOGIN PAGE
   Design a simple login screen centered on the page.
   Include:

* StudyFlow logo at the top
* Email input field
* Password input field
* “Log In” button
* “Create Account” link
* “Forgot Password?” link
  Use a minimal layout with a card-style login box.

2. DASHBOARD PAGE
   This should be the main page users see after logging in.

Layout:

* Left sidebar navigation
* Main dashboard area

Sidebar navigation items:

* Dashboard
* Tasks
* Calendar
* Study Timer
* Progress
* Settings

Dashboard components:

* “Upcoming Deadlines” card showing a list of assignments
* “Today’s Study Tasks” checklist
* “Weekly Study Progress” progress bar or circular chart
* “Quick Add Task” button
* Small calendar widget showing upcoming dates

3. TASK MANAGER PAGE
   Page where users manage their assignments and study tasks.

Include:

* Task list displayed as cards or rows
* Each task shows:

  * Task name
  * Course name
  * Deadline
  * Priority label (High, Medium, Low)
  * Completion checkbox
* Filters at the top:

  * All Tasks
  * Completed
  * Upcoming
* “Add Task” button that opens a task creation form

Task creation form fields:

* Task name
* Course
* Deadline date picker
* Priority dropdown
* Notes field
* Save button
* Cancel button

4. CALENDAR PAGE
   A monthly calendar layout that displays tasks and study sessions.

Include:

* Monthly grid calendar
* Tasks displayed on their deadline dates
* Button to add a new study session
* Clicking a date opens a small modal to add a study session
* Color-coded events for assignments vs study sessions

5. STUDY TIMER PAGE
   A Pomodoro-style study timer.

Include:

* Large circular timer
* Start button
* Pause button
* Reset button
* Timer modes:

  * 25-minute focus session
  * 5-minute break
* Session progress indicator
* Text showing “Current Session: Focus”

6. PROGRESS PAGE
   Shows productivity analytics.

Include:

* Weekly study hours chart
* Completed tasks count
* Study streak counter
* Progress toward weekly study goal
* Visual graphs or bar charts

7. SETTINGS PAGE
   User profile and preferences.

Include:

* Profile section (name, email)
* Study goal settings (hours per week)
* Notification settings
* Theme toggle (light/dark mode)
* Save settings button

Design Requirements:

* Use a consistent navigation sidebar across all pages
* Use card-based layouts for information
* Maintain visual hierarchy with headings and spacing
* Use recognizable icons for navigation
* Ensure all buttons are clearly labeled
* Use clear feedback states such as checked tasks and progress updates

The prototype should include multiple frames for each page and demonstrate clear user flows between:
Login → Dashboard → Add Task → View Calendar → Start Study Timer → View Progress.

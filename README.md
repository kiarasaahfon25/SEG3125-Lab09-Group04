
  # StudyFlow 2

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server (Vite + Express API).

  Requires **Node.js 22.5+** (built-in `node:sqlite`). The API stores data in `data/studyflow.db` (created on first run). Optional: set `SQLITE_PATH` in `.env` to use another file path.
